Codi font (fitxers .java)

  • BeWater.java: Programa principal de simulació de xarxes de distribució d'aigua
  • Canonada.java: Canonada de la xarxa de distribució d'aigua
  • Connexio.java: Node de connexió d'una xarxa de distribució d'aigua
  • Coordenades.java: Coordenades geogràfiques (latitud, longitud)
  • GestorXarxes.java: Mòdul funcional amb funcions per a la gestió de xarxes de distribució d'aigua
  • NodeAixeta.java: Node d'una xarxa de distribució d'aigua
  • Origen.java: Node origen d'una xarxa de distribució d'aigua
  • SimuladorModeText.java: Simula les operacions de construcció, modificació i consulta d'una xarxa de distribució d'aigua a partir d'un fitxer de text.
  • Terminal.java: Node Terminal d'una Xarxa de distribució d'aigua
  • Xarxa.java: Xarxa de distribució d'aigua, no necessàriament connexa (graf dirigit de Node)
