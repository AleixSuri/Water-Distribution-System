Retorna un set amb tots els nodes aixetes els quals estàn trencats o embossats (no passa aigua)

Precondició
Tots els terminals de aiguaArriba pertanyen a la xarxa x, aiguaArriba.get(t) indica si arriba aigua a t i la xarxa x té forma d'arbre
Postcondició
Retorna el conjunt de nodes n de la xarxa x més propers (seguint la topologia) als terminals t de aiguaArriba, tals que per sota de n la situació actual de la xarxa és incoherent amb aiguaArriba
